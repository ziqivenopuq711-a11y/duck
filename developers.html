Getting Started with DuckChain

Setting up your DuckChain development environment takes less than ten minutes. Start by installing the DuckChain CLI using npm install -g @duckchain/cli. Next, initialize a new project with duckchain init my-project and follow the prompts to configure your development environment. Connect to the DuckChain testnet using the built-in faucet to receive free test DUCK tokens for gas fees.

The DuckChain testnet mirrors the mainnet environment with 100% API compatibility, so your code will work without modifications when you deploy to production. Full documentation, interactive tutorials, and a growing library of example projects are available at the official DuckChain developer portal, making it straightforward for both blockchain veterans and newcomers to get productive quickly. The AI Unchained hackathon team hosts live onboarding sessions at the start of the event to help teams get their environments set up and their first transactions on-chain.

Quick start: npm install -g @duckchain/cli && duckchain init my-project && duckchain faucet — you'll be sending your first AI-triggered transaction within minutes.

Smart Contract Development

DuckChain is fully compatible with Solidity and Vyper smart contracts, making it easy for Ethereum developers to migrate or extend their existing projects. DuckChain adds AI-specific precompiles that allow contracts to request inference from registered AI models, store and retrieve vector embeddings on-chain, and interact with AI oracle networks.

The DuckChain Hardhat plugin and Foundry integration make testing and deployment workflows familiar and efficient. For AI Unchained hackathon participants, DuckChain provides a set of audited base contracts including AIAgent.sol, DeFAIPool.sol, and OnChainOracle.sol that teams can extend with their own logic to reduce development time and security risk. These base contracts have been reviewed by leading smart contract security firms and implement industry-standard security patterns, giving hackathon teams a solid and trustworthy foundation to build upon.

AI Agent Framework

Building autonomous AI agents that interact with DuckChain requires careful consideration of both the AI decision-making layer and the blockchain execution layer. DuckChain's Agent SDK provides a unified interface that abstracts the complexity of on-chain interactions, allowing AI models to sign transactions, query contract state, and respond to on-chain events without requiring deep blockchain expertise.

The SDK integrates natively with popular AI frameworks including LangChain, AutoGPT, and ElizaOS. For more advanced use cases, DuckChain's MPC (Multi-Party Computation) wallet infrastructure allows AI agents to control funds without any single party holding the private key, enabling truly trustless autonomous operation. This means an AI agent can manage significant on-chain assets with cryptographic guarantees that no single node, operator, or attacker can unilaterally drain the wallet or halt the agent's operation.

API Reference & Documentation

DuckChain maintains comprehensive developer documentation covering every aspect of the platform. The REST API and GraphQL API allow applications to query chain data, submit transactions, and monitor events programmatically. WebSocket subscriptions provide real-time event streams for responsive application UIs.

The AI Model Registry API allows developers to register, version, and monetize AI models that can be called by smart contracts. All APIs follow OpenAPI 3.0 specifications and include interactive playgrounds in the documentation. Regular changelog updates, migration guides, and breaking change notices ensure developers can stay current with platform evolution. DuckChain's API versioning policy guarantees at least six months of backward compatibility for all stable API versions, so hackathon projects built today will continue to function well into the future.
